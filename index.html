Some of my Projects

Project 1: Speed Typing Game

A typing game that requires the user to type the words shown on screen before they reach the red line.


Utilized queues after randomizing a text file of nouns to store points and display the words


Implemented user input that would cause the cannon to rotate, fire at the word, and remove the letter typed

Project 2: Platform Game

A platform game inspired by Super Mario Bros that requires the user to jump between platforms and dodge obstacles


Designed a level difficulty system that gradually increases the speed and spawn-rate of obstacles as well as lowering the jump height over time


Designed the physics and mechanics behind the game; added gravity to make jumping feel more natural


Implemented a top scores system by writing scores into a text file and then sorting it

Project 3: Maze-Solving Robot

A self-driving car that scans through a maze once, and then takes the shortest path through the maze on its second run


Implemented the brain of the system: Store turns as directions into an array while on its first run, then simplify the array to produce the shortest path possible
